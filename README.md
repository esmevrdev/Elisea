# Elisea
## Dependencias para el proyecto
- [bun](https://bun.com/)

## Primera vez clonando el proyecto
Correr el siguiente comando
```bash
bun install
```
## Correr el proyecto
### General para navegador
```bash
bun start
```
```bash
bun web
```
### IOS
```bash
bun android
```
### Android
```bash
bun android
```
